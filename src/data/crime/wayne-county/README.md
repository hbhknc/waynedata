# Wayne County crime datasets

This directory now includes three official county annual rate series pulled from Table 6 of NCSBI's Crime in North Carolina annual summaries:

- violent-crime
- index-crime-rate
- property-crime-rate

Current coverage in the repo:
- 2007 through 2024 annual county rates
- 2018 explicitly flagged as partial-year data for over 50 percent of Wayne County's population
- shared metadata, summary, and page template wiring

Next historical-extension target:
- add pre-2007 county annual summary values from the historical archive so long-run context can extend further back.
